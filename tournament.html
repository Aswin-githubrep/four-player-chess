⚙ Supabase Setup (one-time)

1
Create free Supabase project

Go to supabase.com → New project → give it a name → set a database password → Create.

2
Run the SQL setup

In Supabase → SQL Editor → paste the SQL from the box below → Run.

3
Paste your project keys

Settings → API → copy Project URL and anon public key.
