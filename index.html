Firebase Setup (one-time)

📋 Step-by-step — takes about 5 minutes

1
Create a free Firebase project

Go to console.firebase.google.com → Add project → enter any name → click Continue (disable Analytics if asked) → Create project.

2
Add a Realtime Database

In the left sidebar → Build → Realtime Database → Create Database → choose any region → Start in test mode → Enable.

3
Copy your database URL

On the Realtime Database page, copy the URL shown at the top. It looks like:
https://your-project-default-rtdb.firebaseio.com

4
Paste it below and connect

That's it — no API keys needed in test mode. Your URL stays saved in this browser.
